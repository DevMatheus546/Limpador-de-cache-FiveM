# 🗑️ Limpador de Cache FiveM

![Windows](https://img.shields.io/badge/OS-Windows-blue)
![Python](https://img.shields.io/badge/Python-3.x-yellow.svg)
![License](https://img.shields.io/badge/License-MIT-orange.svg)

Um aplicativo leve, seguro e com interface gráfica moderna (Dark Mode) desenvolvido para limpar o cache do **FiveM** de forma seletiva. Resolve problemas de textura, falhas de conexão e travamentos em servidores de GTA V Roleplay com apenas um clique.

---

## ✨ Funcionalidades

- **Pronto para uso:** Não precisa instalar nada, basta baixar o `.exe` e rodar.
- **Detecção Automática:** Encontra automaticamente a pasta de cache do FiveM no seu computador.
- **Limpeza Seletiva:** Permite escolher exatamente quais pastas apagar:
  - `cache`
  - `game-storage`
  - `nui-storage`
  - `server-cache`
  - `server-cache-priv`
- **Opção Avançada:** Opção dedicada para apagar as binds do GTA (`fivem.cfg`).
- **Segurança Antifalhas:** Se o GTA estiver aberto durante a limpeza, o aplicativo não irá "crashar", mas avisará quais arquivos não puderam ser deletados.
